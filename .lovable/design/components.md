# Components

## From `@nvidia/foundations-react-core`
- `ThemeProvider` — Wraps app with NVIDIA theme
- `Text` — Typography component. Props: `kind` (title/lg, title/md, body/sm, label/md, etc.)
- `AppBar` — Top navigation bar. Props: `slotLeft`, `slotRight`
- `Avatar` — User avatar. Props: `size`, `fallback`
- `Stack` — Flex layout
- `Anchor` — Link component
- `VerticalNav`, `VerticalNavLink`, `VerticalNavText` — Side navigation

## Custom
- `DesignSystemLayout` — Layout with AppBar + VerticalNav sidebar (src/components/layout/)
